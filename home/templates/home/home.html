User Manual

User Authentication

  • Login
  • Logout

Paper Page

  • Go to the Paper List Page.
  • Input search words and press Search button.
  • Click Add button at the bottom to add your paper information manually.

Topic Page

  • 주제 목록 화면으로 이동한다.
  • To add a new topic, click the Create New Topic button at the bottom.
  • Enter the sentences to search for in Keywords, separated by commas (",").
  • Clicking on a topic on the Topic List Page takes you to the topic's Details Page.

Topic's Detail Page

  • Clicking the Collect Arxiv button collects the specified number of papers.
  • Clicking the All ↺ button to get citation information for all the paper.
  • You can view PDF(), Note(), Google scholar(), and Semantic scholar().

Note Page

  • There is the PDF View, the Note Editor, and the Markdown Preview.
  • Clicking the Import PDF button extracts text from the PDF.
  • Clicking the Summarize button organizes the contents of the note editor.
  • The summarized contents are added to the end of the note editor.

논문 목록 화면

  • 논문 목록 화면으로 이동한다.
  • 검색할 단어를 입력하고 Search 버튼을 누른다.
  • 논문 정보를 수동으로 입력하려면 화면 하단의 Add 버튼을 누른다.

주제 목록 화면

  • 주제 목록 화면으로 이동한다.
  • 새 주제를 추가하려면 화면 하단의 Create New Topic 버튼을 누른다.
  • 새 주제 추가 화면의 Keywords에 검색 문장을 쉼표(",")로 구분하여 입력한다.
  • 주제 목록 화면에서 주제를 클릭하면 해당 주제의 상세 화면으로 이동한다.

주제 상세 화면

  • Collect Arxiv 버튼을 누르면 지정한 갯수만큼 논문을 수집한다.
  • All ↺ 버튼을 누르면 논문의 인용 수 정보를 얻는다.
  • PDF(), 노트(), Google scholar(), Semantic scholar()를 볼 수 있다.

노트 화면

  • 좌측부터 PDF 보기, 노트 편집기, 마크다운 미리보기 화면이다.
  • Import PDF 버튼으로 PDF에서 텍스트를 추출하여 노트 편집기에 출력한다.
  • Summarize 버튼으로 노트 편집 화면의 내용을 Google Gemini로 정리한다.
  • 요약 정리된 내용은 노트 편집 화면의 하단에 추가된다.
